This board is a minimalist development platform for Xilinx Spartan-6 FPGAs in TQ144 package (more specifically XC6SLX4-*TQG144* and XC6SLX9-*TQG144*).

Peripherals / connectors:
	Standard 2x7mm Molex header for JTAG
	10-pin FFC for JTAG (my low-profile standard connector)
	FT232RQ USB-serial adapter
	Eight 0805 LEDs
	Reset button
	20 MHz CMOS oscillator
	20-pin FFC for GPIO
	Probe/test pads on all possible pins not brought out to GPIO headers
